# Plantilla
## ✨ Vite+Electron+TypeScript+Vue ✨

### Instalación

```npm install```
### Compilación + execución

```npm run build```

```npm start```

Nota: importante compilar antes de executar, e dentro da carpeta, para iso:

```cd vite-electron-vue-ts project```

### Testeo

```npm run test```

### Outros
Se se quere abrir no navegador con Vite:
```npm run dev```
